# Estate Network – Premium Real Estate Platform

A **full-stack real estate marketplace** built with MERN stack (MongoDB, Express, React, Node.js) featuring property listings, secure authentication, admin dashboard, real email notifications, image uploads, and responsive design.

**Demo Video:** https://youtu.be/ZgKTtgh48uY ||
 **Live Demo:** currently working on some functionality || 
 **Backend:** currently working on some functionality 

---

### Features

- User & Admin Authentication (JWT + httpOnly Cookies)
- Create, Edit, Delete Properties with Multiple Image Upload
- Advanced Property Search & Filters
- Contact Property Owner (Real Gmail Emails via Nodemailer)
- Admin Dashboard with Stats & Contact Management
- Discounted Price Support (including 0 for free properties)
- Fully Responsive Design (Mobile + Desktop)
- Dark Mode + Beautiful UI with MUI & Tailwind-like styling
- Secure File Uploads with Cloudinary
- Real-time Form Validation (Yup + React Hook Form)

---

### Tech Stack

| Layer         | Technology                              |
|-------------|-----------------------------------------|
| Frontend     | React, Vite, Redux Toolkit, MUI, Axios |
| Backend      | Node.js, Express, MongoDB, Mongoose     |
| Auth         | JWT, httpOnly Cookies                   |
| Email        | Nodemailer + Gmail App Password         |
| File Upload  | Multer + Cloudinary                     |
| Deployment   | Vercel (Frontend) • Render (Backend)    |
